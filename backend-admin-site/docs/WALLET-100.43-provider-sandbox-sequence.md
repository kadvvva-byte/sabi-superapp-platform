# WALLET-100.43 — Provider Sandbox Sequence

Recommended sandbox sequence:

1. KYC provider sandbox
2. AML provider sandbox
3. Bank card tokenization sandbox
4. Local bank gateway sandbox
5. Virtual card issuer sandbox
6. Alipay+ acquiring sandbox
7. Merchant acquiring sandbox
8. Business payout sandbox
9. Coin Wallet ledger/treasury sandbox
10. Crypto custody sandbox
11. Crypto market data sandbox

Execution flags must remain disabled until the matching provider adapter is bound to a real sandbox provider and returns verifiable provider references.
