# WALLET-100.59 — Coin/Crypto env requirements

Required future env / vault refs:

Coin ledger / treasury:
- WALLET_PROVIDER_COIN_WALLET_LEDGER_ENABLED=true only after real provider binding
- WALLET_PROVIDER_COIN_WALLET_LEDGER_ID
- WALLET_PROVIDER_COIN_WALLET_LEDGER_BASE_URL
- WALLET_PROVIDER_COIN_WALLET_LEDGER_API_KEY_VAULT_REF
- WALLET_PROVIDER_COIN_WALLET_LEDGER_WEBHOOK_SECRET_VAULT_REF
- WALLET_PROVIDER_COIN_WALLET_LEDGER_WEBHOOK_URL
- WALLET_PROVIDER_EXECUTION_COIN_WALLET_LEDGER_ENABLED=true only after real adapter verification

Crypto custody:
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_ENABLED=true only after real custody binding
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_ID
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_BASE_URL
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_API_KEY_VAULT_REF
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_WEBHOOK_SECRET_VAULT_REF
- WALLET_PROVIDER_CRYPTO_CUSTODY_PROVIDER_WEBHOOK_URL
- WALLET_PROVIDER_EXECUTION_CRYPTO_CUSTODY_PROVIDER_ENABLED=true only after real adapter verification

Crypto market data:
- WALLET_PROVIDER_CRYPTO_MARKET_DATA_PROVIDER_ENABLED=true only after real market data provider binding
- WALLET_PROVIDER_CRYPTO_MARKET_DATA_PROVIDER_ID
- WALLET_PROVIDER_CRYPTO_MARKET_DATA_PROVIDER_BASE_URL
- WALLET_PROVIDER_CRYPTO_MARKET_DATA_PROVIDER_API_KEY_VAULT_REF
- WALLET_PROVIDER_EXECUTION_CRYPTO_MARKET_DATA_PROVIDER_ENABLED=true only after real adapter verification

Never place raw provider API keys, private keys, seed phrases, or webhook secret values in mobile.
